# SouEdu

**SouEdu** é um projeto de páginas estáticas (GitHub Pages) com dicas sobre como utilizar **licenças de software** — permanentes ou temporárias — concedidas a alunos de instituições de ensino, especialmente de nível superior ou profissionalizante.

## Acesse o site

👉 [https://kelsoncm.github.io/SouEdu/](https://kelsoncm.github.io/SouEdu/)

## Conteúdo

- **Início** – Visão geral sobre licenças educacionais
- **Licenças Permanentes** – Dicas de uso para licenças que não expiram após a formatura
- **Licenças Temporárias** – Dicas de uso para licenças válidas somente durante o vínculo acadêmico
